🎯 WORKFLOW OVERVIEW:

Customer Journey:
  1. Submit pricing request → Confirmation email ✅
  2. Wait for review → Admin notification sent ✅
  3. Receive approval → Approval email ✅
  4. Get payment link → Payment link email ✅
  5. Make payment → Confirmation email ✅
  6. Plan activated! → Status updated ✅

Admin Workflow:
  1. View pending requests (dashboard)
  2. Review requirements
  3. Click "Approve" or "Reject"
  4. If Approve: Create pricing plan
  5. Create Razorpay order → Customer gets payment link
  6. Track payment status
  7. Done! (Emails sent automatically)
